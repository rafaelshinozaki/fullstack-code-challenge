Usage Tips
Always paste the full context in the first prompt (Notion requirements + JSON)
Don't skip prompts — the order matters because each one builds on the previous
Review before moving on — if something looks off, ask for adjustments before the next prompt
Prompts 1–10 are the core — prompts 11 and 12 are complementary but demonstrate engineering maturity
If the conversation gets too long, you can start a new one and reference what's already built with:
"I already have files X, Y, Z implemented"
